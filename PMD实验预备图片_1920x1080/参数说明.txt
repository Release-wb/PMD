PMD 实验预备图片参数说明

【相机与镜头】
相机型号: MER2-502-79U3C
相机分辨率: 2448 x 2048 px
相机像元尺寸: 3.45 um x 3.45 um
镜头型号: HN-2516-5M-C2/3X
镜头焦距: 25 mm
按像元尺寸估算焦距: f ≈ 7246.38 px

【LCD】
LCD 图片分辨率: 1920 x 1080 px
LCD 有效显示区域: 344.000 mm x 194.000 mm
LCD 水平像素物理尺寸 pp_x = 0.179166667 mm/px
LCD 垂直像素物理尺寸 pp_y = 0.179629630 mm/px

【条纹】
四步相移灰度范围: 32 到 222
条纹周期 T = 64 px
X 方向周期数 ceil(1920/T) = 30, Gray Code 位数 = 5
Y 方向周期数 ceil(1080/T) = 17, Gray Code 位数 = 5
Gray Code 已生成原码和反码，建议都采集，用反码做阈值鲁棒判断。

【屏幕外参标定图】
棋盘格: 13 x 8 个方格, 内角点: 12 x 7, 方格边长约 20.000 mm, 像素尺寸约 112 x 111 px
对称圆点阵列: 13 x 7, 圆心间距约 20.000 mm, 像素间距约 112 x 111 px, 圆半径约 16 px

【实验提醒】
1. 显示图片时请关闭缩放、护眼、HDR、自动亮度、色彩增强，确保 1:1 像素显示。
2. PMD 采集时相机焦距、光圈、曝光时间、增益、白平衡必须锁定。
3. 条纹图不要过曝，建议先用中灰图和四步条纹第1步检查直方图。
4. 相机不必拍到整个 LCD，只要有效测量区域内能看到完整编码且能解出绝对相位即可。
5. 第一次实验建议先采参考平面镜两遍，做参考-参考差分，确认重建接近零。
